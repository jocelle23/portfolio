About Me

When I'm not coding, I'm out and about with friends and family. With a family of foodies, there's always new places to explore. You can also catch me playing games online with friends. Fun fact: I'm a dog mom to a pomsky named Kota!

Web development is an area I've had an interest in for years. I've tried teaching myself code back in high school, but I didn't set aside enough time to really nail core concepts. I eventually got discouraged and stopped coding altogether. After joining Thinkful's Software Engineering program, I've realized that learning code is like learning another language. It takes time, patience, and lots of practice, but it's worth it! Building awesome projects with lines of code used in our ever day language has been a challenging but rewarding experience. Throughout my time in Thinkful's SE program, I've realized my passion for coding, and I'm ready to collaborate on projects with other great programmers!
